## 方案一：使用自定义`EnvironmentKey`
适合`SwiftUI`结构内传递、统一配置

1️⃣ 定义一个 EnvironmentKey 和扩展：
```Swift
struct AppURLKey: EnvironmentKey {
  static let defaultValue = URL(string: "http://default.example.com")!
}


extension EnvironmentValues {
  var appURL: URL {
    get { self[AppURLKey.self] }
    set { self[AppURLKey.self] = newValue }
  }
}
```

2️⃣ 使用时注入当前环境的值：

例如在`App`中注入：
```swift
@main MyApp: App {
  var body: some Scene {
    let environmentURL = URL(string: "https://dev.example.com")!
    WindowGroup {
      ContentView()
        .environment(\.appURL, environmentURL)
    }
  }
}
```

3️⃣ 在视图中使用：
```swift
struct ContentView: View {
  @Environment(\.appURL) var appURL

  var body: some View {
    Text("App URL: \(appURL.absoluteString)")
  }
}
```
